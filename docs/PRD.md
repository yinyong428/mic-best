> Version 1.0 | 2026-04-05 | Olly

## 一、产品概述

### 1.1 使命

让任何人——从Maker到工程师——通过自然语言对话，在几分钟内完成硬件原型设计。

不再需要手动查阅数据手册、不再需要对着Excel表配零件、不再需要用Visio画接线图。**说你想要的，AI帮你设计。**

### 1.2 核心价值主张

| 用户痛点 | 我们的解决方案 |
|---------|--------------|
| "我不知道用什么零件" | AI 根据需求推荐并生成完整 BOM |
| "接线图太复杂容易出错" | React Flow 可视化接线，节点类型化 |
| "3D模型不知道零件位置" | Three.js 爆炸图，装配可视化 |
| "组装步骤无从下手" | AI 生成详细分步装配指南 |
| "估算成本不清楚" | 自动计算总价 + Amazon 购买链接 |

### 1.3 目标用户

**Primary:**
- Maker / 硬件创客（树莓派、Arduino 项目）
- 硬件创业团队（快速原型验证）
- 电子爱好者（学习硬件设计）

**Secondary:**
- 产品经理（硬件产品概念验证）
- 教育场景（STEM 教具设计）
