## 1. 执行摘要（Executive Summary）

**MIC Blueprint** 是中国版 AI 硬件原型设计平台，由"中国制造"供应链深度驱动。用户用自然语言（优先中文）描述想法，AI 在数秒内生成**完整、可采购、可组装的硬件原型**，默认对接**立创商城（LCSC）+ JLCPCB + 嘉立创EDA** 供应链体系，实现**比国外 Blueprint 更低成本、更快交付**。

核心差异：用中国供应链 **30 秒生成硬件蓝图**，同时服务国内创客、学生、硬件团队，并反向输出全球市场。

## 2. 产品定位与战略价值

### 2.1 为什么现在做？

**Blueprint.am 的价值已被验证**：
- 3E8 Robotics 打造的 Blueprint.am 证明了 AI 生成硬件原型的市场需求真实存在
- 核心功能（ wiring diagram + BOM + assembly guide from prompt）已跑通
- 但 Blueprint.am 依赖 Digi-Key/Mouser，价格高、交付慢（美国仓库）

**中国供应链的天然优势**：
- LCSC 零件比 Digi-Key 便宜 30-70%，部分零件便宜 90%+
- JLCPCB PCB 打样 5 片仅 $2，SMT 贴片起价 $0
- 嘉立创生态（EDA + 商城 + PCB + SMT + 3D 打印）全覆盖
- 人民币计价，消除汇率风险

### 2.2 战略定位

```
海外创客 → 用中国供应链低成本设计 → MIC Blueprint
     ↑                                     ↓
     └──────── 反向输出全球市场 ← 中文内容社交传播
```

**不是"复制 Blueprint"，是"用中国供应链重新定义"**：同样的 AI 能力，更低的零件成本、更快的交付、更本土的体验。

### 2.3 竞争优势对比

| 功能维度 | Blueprint.am（美国原版） | MIC Blueprint（中国版） |
|---------|------------------------|----------------------|
| 零件库 | Digi-Key / Mouser（贵） | LCSC 为主（便宜 30-70%） |
| 计价货币 | USD | RMB / USD 双轨 |
| PCB 打样 | 第三方工厂 | JLCPCB（$2/5片） |
| BOM 成本 | 高 | 低 |
| 本地化 | 英文 | 中文优先 + 英 |
| 支付 | Stripe | 微信/支付宝/支付宝国际 |
| 社交分享 | Twitter/YouTube | 小红书/B站/抖音 |
| 大模型 | GPT-4o / Claude | 通义千问 + DeepSeek 双引擎 |

## 3. 目标用户与人物画像

### 3.1 主要用户群

**用户群 A：国内硬件创客 / 学生（70%）**
- 典型：电子类专业本科生、硬件初创公司技术负责人、开源硬件爱好者
- 痛点：不知道选什么零件、不会画原理图、不知道从哪里采购便宜
- 场景：课程设计、毕业设计、智能硬件 MVP
- 付费意愿：中低（学生），中高（创客团队）
- 平台偏好：微信、B站、小红书

**用户群 B：海外创客（30%）**
- 典型：美国/欧洲个人创客、开源硬件社区
- 痛点：Digi-Key 太贵，交付太慢，想要便宜零件
- 场景：ESP32 项目、Arduino 原型、机器人
- 付费意愿：中高（Pro 订阅）
- 平台偏好：Twitter、YouTube、GitHub

**用户群 C：中小企业 / 教育机构（10%）**
- 典型：硬件创业公司硬件工程师、中专/职校电子专业
- 痛点：重复选型耗时，BOM 管理混乱，采购效率低
- 场景：产品原型设计、实验教学

### 3.2 用户旅程（完整路径）

```
[用户输入自然语言需求]
       ↓
[AI 解析 → 生成多 Tab 项目]
  ├── Chat（对话迭代）
  ├── Info（项目信息 / 规格）
  ├── BOM（物料清单 + 实时价格）
  ├── Wiring（接线图）
  ├── Mech（机械结构 / 3D 视图）
  └── Instructions（组装步骤）
       ↓
[一键导出：LCSC 购物车 + JLCPCB Gerber]
       ↓
[用户下单 → 零件到手 → 组装]
       ↓
[社区分享：小红书 / B站 / GitHub]
```

## 4. 核心功能需求（Feature List）

### 4.1 多 Tab 工作区

| Tab 名称 | 功能描述 | 核心价值 |
|---------|---------|---------|
| **Chat** | 自然语言对话，支持中文迭代 | 零门槛沟通需求 |
| **Info** | 项目概述：名称、描述、主控芯片、外设列表 | 结构化需求确认 |
| **BOM** | 物料清单，含 LCSC 零件编号、实时价格、库存状态 | 一键采购 |
| **Wiring** | 接线图（Fritzing 风格 / SVG 交互图） | 消除接线错误 |
| **Mech** | 机械结构图（STEP / STL 预览 + 装配说明） | 3D 可视化 |
| **Instructions** | 分步组装指南（含注意事项） | 快速上手 |

### 4.2 AI 对话与迭代

- **自然语言输入**：支持复杂中文需求
  - 示例："用 ESP32-S3 做一个温湿度监控系统，带 OLED 显示屏和 Wi-Fi 上传功能，需要低功耗设计"
- **多轮迭代**：对话中修改
  - 示例："把 OLED 换成电子墨水屏" / "增加摄像头识别并接入通义千问"
- **实时预览**：每次迭代自动更新所有 Tab
- **中英双语输出**：UI 中文，生成的 BOM/规格书可切换英文

### 4.3 中国供应链核心功能（v2.0 差异化）

#### 4.3.1 BOM 模块（重点）

| 功能 | 描述 |
|------|------|
| **LCSC 实时价格** | 默认展示 LCSC 零件编号、单价（RMB/USD）、库存状态 |
| **一键切换供应商** | LCSC 无货 → 自动 fallback 到 Digi-Key / Mouser |
| **1688 采购清单** | 大批量采购一键生成 1688 批发链接 |
| **淘宝联盟推广链接** | 生成带联盟 PID 的淘宝/天猫商品链接（平台可获佣金分成） |
| **JLCPCB 基础零件** | 自动标记 JLCPCB SMT 贴片零件，优先选用 |
| **运费 / 关税估算** | 基于重量体积的国内快递费估算，含进口关税提示 |
| **BOM 导出** | CSV/Excel 格式，可直接导入 LCSC 购物车 |

#### 4.3.2 PCB 导出模块

| 功能 | 描述 |
|------|------|
| **Gerber 文件生成** | 从 Wiring 图自动生成标准 Gerber 文件 |
| **JLCPCB 自动上传** | 一键将 Gerber + BOM 上传至 JLCPCB 报价系统 |
| **SMT 贴片报价** | 实时 SMT 加工费计算（基于 BOM 中 JLC 基础零件） |
| **嘉立创EDA 导出** | 导出 .esym 格式，直接在嘉立创EDA打开编辑 |
| **3D 打印件清单** | Mech 模块中的 3D 打印件，自动推荐创想三维/闪电打印 |

#### 4.3.3 零件库与选型

- **国产主控优先推荐**：
  - ESP32 系列（乐鑫）
  - 香橙派（Orange Pi）系列
  - 全志（Allwinner）系列
  - 瑞芯微（RK3588）系列
  - STM32 国产替代（GD32 / CH32）
- **LCSC 热门零件缓存**：每 5 分钟同步一次热门零件价格和库存
- **零件替代建议**：当指定零件无货时，AI 自动推荐功能等效替代品

## 7. 风险与缓解

| 风险 | 概率 | 影响 | 缓解方案 |
|------|------|------|---------|
| LCSC API 申请被拒 | 中 | 高 | 提前准备企业资质，同时对接 Digi-Key API 作为 fallback |
| JLCPCB API 审核周期长 | 高 | 中 | Phase 1 使用手动 Gerber 导出，Phase 2 再集成 API |
| AI 生成接线图错误 | 中 | 高 | 引入规则引擎校验 + 用户确认机制 |
| 供应链价格波动 | 高 | 中 | 缓存价格快照（5 分钟有效期），下单前强制确认 |
| 零件库存不准确 | 中 | 中 | 实时库存状态显示「充足 / 紧张 / 缺货」，引导替代 |
| 淘宝联盟 API 申请限制 | 中 | 低 | 备用京东联盟 / 多多客 |
